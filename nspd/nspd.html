Theory of Operation

AOLserver's nsext driver connects to databases through a database proxy daemon.  That proxy daemon uses code that is in libnspd.a to talk with AOLserver.

Until we revise the documentation for placement here, please see the Database Driver notes at http://aolserver.com/doc/.
