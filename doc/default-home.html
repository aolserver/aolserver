  • Fast Multithreading On All Platforms
  • Powerful, Multithreaded C and Tcl APIs
  • Dynamic Scripting with AOLserver Dynamic Pages
  • Extensible Architecture With AOLserver Modules
  • Direct Access To Sybase, Postgres, SOLID, Interbase, Informix, MySQL, Oracle, And More
  • Comprehensive Online Documentation
  • The AOLserver Community System
  • And Much More...
