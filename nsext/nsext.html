Theory of Operation

This driver is used to interact with external database drivers.  These are usually called "database proxy daemons".

Known Issues

The driver API talks over sockets to remote daemons in an inefficient manner.  It is highly recommended that the proxy daemon and AOLserver both run on the same machine.  Of course, the database can still run on a remote machine -- it's only the connection between AOLserver and the proxy daemon that is inefficient and that inefficiency is greatly alleviated by running on the same host.
